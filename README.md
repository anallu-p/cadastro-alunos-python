#  Sistema de Cadastro de Alunos

Sistema desenvolvido em Python para gerenciamento de cadastro de alunos,
com foco em estruturas de dados e programação modular.

##  Funcionalidades

- Cadastrar novos alunos
- Consultar alunos registrados
- Organizar e listar dados
- Estrutura modular e organizada

##  Tecnologias utilizadas

- Python 3
- Lógica de programação
- Estruturas de dados
- Programação modular

## ▶️ Como executar

1. Clone o repositório:
   git clone https://github.com/anallu-p/cadastro-alunos-python

2. Acesse a pasta do projeto:
   cd cadastro-alunos-python

3. Execute o arquivo principal:
   python main.py
